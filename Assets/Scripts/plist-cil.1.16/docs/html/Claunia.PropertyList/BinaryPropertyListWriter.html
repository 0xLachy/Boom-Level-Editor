BinaryPropertyListWriter Class

A BinaryPropertyListWriter is a helper class for writing out binary property list files.

It contains an output stream and various structures for keeping track of which NSObjects have already been serialized, and where they were put in the file.

ReuseObjectIds Property

Gets or sets a value indicating whether two equivalent objects should be serialized once in the binary property list file, or whether the value should be stored multiple times in the binary property list file. The default is false.

Syntax

public bool ReuseObjectIds { get; set; }

Value

Documentation for this section has not yet been entered.

Remarks

In most scenarios, you want this to be true, as it reduces the size of the binary proeprty list file. However, by default, the Apple tools do not seem to implement this optimization, so set this value to false if you want to maintain binary compatibility with the Apple tools.
